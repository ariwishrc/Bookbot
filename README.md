# 📚 Bookbot
My Approach for Bookbot project from boot.dev

<img width="471" height="682" alt="Screenshot_20260611_025248" src="https://github.com/user-attachments/assets/0ecd4d80-d16f-4238-b01c-974909c18bb2" />

Bookbot is a simple python script to count the words and letters from a text file. I also added the percentage of each letter.

Currently there is 2 books in Books/ (for testing purposes)
- Vigilante.txt
- The_Space_Saga_of_Rocket.txt

# How to Run
1. Make sure python is installed
2. run ```python bookbot.py books/Vigilante.txt```
